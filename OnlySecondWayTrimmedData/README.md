# Ecg
В оригтнальном датасете есть 12 отведений.
Наш компактный девайс умеет снимать ЭКГ по отведению №2. Соотвтетсвенно задача здесь - это обучать модель именно данными только 2ого из 12 отведений. 
Модель, уже обученная на триммированных данных - model.hdf5, по ссылке https://disk.yandex.ru/d/1a4XxA710dX9LA

Тестовый набор данных - https://disk.yandex.ru/d/YFsOwReoL3VGug
gold_standard.csv содержит корректную и правлиьную информацию о каждом из 857 сэмплов тестовой выборки из файла выше.

Оригинальный кусок данных для обучения (с 12 отведениями) весит более 3Гб.
